# Emmet语法

## 认识Emmet语法

Emmet（前身为Zen Coding）是一个能大幅度`提高前端开发效率`的一个工具。

- Vscode内容了Emmet语法
- 高效输入HTML和CSS语法

## 具体语法

### HTML

1. `!和html:5`：快速生成HTML5框架

2. `>` ： 子代

3. `+` ： 兄弟

   div>h1+p

4. `*`：个数

5. `^ `：上一级

6. `()` ：分组

7. `[]`：属性

8. `#` ：id

9. `. `：类

10. `{ } `：内容

11.  `$ `：序号，几个 \$就是几位数字

### CSS

1. `w100` ： width: 100px;

2. `w20+h30+m40+p50 `
